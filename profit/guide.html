Tier Transition Matrix (등급 전환 매트릭스)

tier_matrix.html  →  http://43.200.4.183/profit/tier_matrix
바로가기 →
페이지 개요
고객들이 연도별로 등급이 어떻게 변화했는지를 매트릭스 형태로 보여주는 페이지입니다. 전년도 등급(FROM) → 당해 등급(TO)의 건수와 행 기준 비율(%)을 5×5 매트릭스로 시각화합니다.
  • 요약 지표 (5개) — 신규 진입(전년 없음), 이탈(당해 없음), 등급 상승, 등급 하락, 등급 유지
  • 건수 매트릭스 — FROM(전년도 상태) × TO(당해 상태)의 고객 수
  • 비율 매트릭스 — 각 행(FROM 등급)에서 TO 등급으로 전환된 비율(%)
  • 등급 상태 — VIP / Gold / Silver / Bronze / Basic / NEW(신규) / CHURN(이탈)
tier_matrix_data.json suite_tier.csv generate_tier_matrix.py 생성
사용 데이터 상세
suite_tier.csv에서 연도별 등급 이력을 읽어 전환 매트릭스를 산출합니다.
컬럼설명
suite_number고객 창고 ID
sign_up_date회원 가입일 (YYYY-MM-DD)
y23_grade2023년 등급 (VIP/Gold/Silver/Bronze/Basic 또는 null)
y24_grade2024년 등급
y25_grade2025년 등급
매트릭스 산출 로직:
— 전년도 등급이 없고 당해 등급이 있으면 → NEW (신규 진입)
— 전년도 등급이 있고 당해 등급이 없으면 → CHURN (이탈)
— 나머지는 FROM 등급 → TO 등급 전환으로 집계

Shipping Fee Simulation (배송비 시뮬레이션)

shipping_fee.html  →  http://43.200.4.183/profit/shipping_fee
바로가기 →
페이지 개요
배송비 단가를 변경했을 때 매출이 얼마나 달라지는지 시뮬레이션하는 도구입니다. 무게 구간별로 DK Count(출고 수량)와 P_DK 단가(USD)를 직접 수정하면 Ship Revenue가 실시간으로 재계산됩니다.
  • 필터 — 국가(Country), 배송사(Carrier), 배송사 서비스(Service), 배송비 결제일 From~To
  • KPI (3개) — Total Packages (QDK), Simulated Ship Revenue (PDK×QDK), We Buy Revenue (구매대행 매출, USD)
  • 무게 구간 테이블 — 무게 kg 구간별: DK Count(수정 가능), P_DK 단가(수정 가능), Ship Revenue, We Buy Revenue, Case별 건수(C1~C5)
활용 방법: 특정 배송사/국가/기간을 필터링한 후 단가를 조정하여 가격 변경 시나리오에서 예상 배송 매출을 즉시 산출할 수 있습니다.

파생 컬럼
시뮬레이션 계산식:
Ship Revenue = Σ (P_DK × Q_DK) (무게 구간별 합산)
P_DK: 사용자가 직접 수정 가능한 단가 (기본값: 실제 결제 단가 평균)
Q_DK: 해당 무게 구간 패키지 수 (수정 가능)

Loyalty Tier (2026년 회원 등급)

loyalty_tier.html  →  http://43.200.4.183/profit/loyalty_tier
바로가기 →
페이지 개요
2025년 실적을 기준으로 산정한 2026년 회원 등급 결과를 보여주는 페이지입니다.
  • 등급 요약 카드 (5개) — VIP / Gold / Silver / Bronze / Basic 각 인원 수
  • 등급 기준 — 전체 사용 금액 + 배송 횟수 5회 이상 조건 기반 5단계 등급
  • Technical Notes — 2025-03-31 이전 가입 + y25 등급 결측 → Basic 자동 부여 처리 내역
  • 2026 vs 2025 등급 변화 — 각 등급별 2026년 인원 수 + 2025년 대비 증감(+/-)
  • 등급별 평균 실적 — 등급별 인원 수, 전체 사용 금액 평균, 구매대행 결제 평균, 배송비 결제 평균, 창고 옵션 결제 평균
  • 전체 고객 테이블 — Suite Number(Customer Search 링크), 수치별 데이터, 2026~2023 등급 이력
