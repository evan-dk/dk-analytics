Customer Search (고객 분석)

customer_profile.html  →  http://43.200.4.183/profit/customer_profile
바로가기 →
페이지 개요
Suite Number(고객 창고 ID)를 입력하면 해당 고객의 전체 수익 분석 결과를 보여주는 페이지입니다. URL 파라미터 ?suite=XXXX로 직접 접근도 가능합니다 (Loyalty Tier 테이블의 Suite 링크 클릭 시 자동 이동).
  • KPI Cards (6개) — 해당 고객의 Total Profit / Total Revenue / Buy Revenue / Storage Revenue / Ship Revenue / Shipped Packages
  • Revenue Split 파이 차트 — Buy / Storage / Ship 매출 비중 시각화
  • Case Usage 막대 차트 — Case 1~5별 사용 건수 빈도
  • Detailed Metrics & Insights — 주요 지표 상세 수치 + Insight Memo
dashboard_data.json all_suites 배열에서 suite_number로 필터
사용 데이터 상세
dashboard_data.json의 all_suites 배열에서 suite_number로 검색합니다. 각 Suite 객체에는 다음 집계값이 포함됩니다:
필드설명
suite_number고객 창고 ID (Suite 번호)
total_revenue_krw총매출 합계 (원화): goods + warehouse + shipping revenue
total_profit_krw순이익 합계 (원화)
buy_revenue_krw구매대행 매출 합계 (goods_revenue_krw, 원화)
storage_revenue_krw창고 서비스 수수료 합계 (warehouse_revenue_krw, 원화)
ship_revenue_krw배송비 매출 합계 (shipping_revenue_krw, 원화)
package_count출고 패키지 수 (건수)
case_countsCase 1~5별 사용 건수 분포
revenue_percentile전체 고객 대비 Total Revenue 백분위수

Full Rankings (고객 전체 랭킹)

customer_ranking.html  →  http://43.200.4.183/profit/customer_ranking
바로가기 →
페이지 개요
전체 활성 고객을 Total Revenue 기준으로 정렬하여 표로 보여주는 페이지입니다. 각 컬럼 헤더 클릭 시 오름/내림차순 정렬, Suite Number 검색, 관리자 계정 제외 토글, CSV 내보내기 기능을 제공합니다.
  • Suite No. — 고객 창고 ID
  • Total Revenue — 총매출 (원화)
  • Buy Revenue — 구매대행 매출 (원화)
  • Storage Rev — 창고 서비스 매출 (원화)
  • Ship Revenue — 배송비 매출 (원화)
  • Total Profit — 순이익 (원화)
  • Packages — 출고 패키지 수
  • Total Revenue Percentile — 전체 대비 백분위수
dashboard_data.json all_suites 배열 전체
사용 데이터 상세
dashboard_data.json의 all_suites 배열 전체를 로드합니다. Customer Search와 동일한 데이터 소스이며, 모든 고객(admin 제외 시 Z9996~Z9999 제외)을 한 화면에 표시합니다.

관리자 계정(Admin) 제외 대상:
Z9996, Z9997, Z9998, Z9999 (대소문자 모두)

Percentile 산출:
전체 고객(admin 제외)의 total_revenue_krw를 오름차순 정렬 후, 각 고객의 순위 ÷ 전체 고객 수 × 100으로 계산합니다.
정렬 가능 컬럼: 8개 50건/페이지 페이지네이션 CSV 내보내기 지원

Tier Transition Matrix (등급 전환 매트릭스)

tier_matrix.html  →  http://43.200.4.183/profit/tier_matrix
바로가기 →
페이지 개요
고객들이 연도별로 등급이 어떻게 변화했는지를 매트릭스 형태로 보여주는 페이지입니다. 전년도 등급(FROM) → 당해 등급(TO)의 건수와 행 기준 비율(%)을 5×5 매트릭스로 시각화합니다.
  • 요약 지표 (5개) — 신규 진입(전년 없음), 이탈(당해 없음), 등급 상승, 등급 하락, 등급 유지
  • 건수 매트릭스 — FROM(전년도 상태) × TO(당해 상태)의 고객 수
  • 비율 매트릭스 — 각 행(FROM 등급)에서 TO 등급으로 전환된 비율(%)
  • 등급 상태 — VIP / Gold / Silver / Bronze / Basic / NEW(신규) / CHURN(이탈)
tier_matrix_data.json suite_tier.csv generate_tier_matrix.py 생성
사용 데이터 상세
suite_tier.csv에서 연도별 등급 이력을 읽어 전환 매트릭스를 산출합니다.
컬럼설명
suite_number고객 창고 ID
sign_up_date회원 가입일 (YYYY-MM-DD)
y23_grade2023년 등급 (VIP/Gold/Silver/Bronze/Basic 또는 null)
y24_grade2024년 등급
y25_grade2025년 등급
매트릭스 산출 로직:
— 전년도 등급이 없고 당해 등급이 있으면 → NEW (신규 진입)
— 전년도 등급이 있고 당해 등급이 없으면 → CHURN (이탈)
— 나머지는 FROM 등급 → TO 등급 전환으로 집계

Shipping Fee Simulation (배송비 시뮬레이션)

shipping_fee.html  →  http://43.200.4.183/profit/shipping_fee
바로가기 →
페이지 개요
배송비 단가를 변경했을 때 매출이 얼마나 달라지는지 시뮬레이션하는 도구입니다. 무게 구간별로 DK Count(출고 수량)와 P_DK 단가(USD)를 직접 수정하면 Ship Revenue가 실시간으로 재계산됩니다.
  • 필터 — 국가(Country), 배송사(Carrier), 배송사 서비스(Service), 배송비 결제일 From~To
  • KPI (3개) — Total Packages (QDK), Simulated Ship Revenue (PDK×QDK), We Buy Revenue (구매대행 매출, USD)
  • 무게 구간 테이블 — 무게 kg 구간별: DK Count(수정 가능), P_DK 단가(수정 가능), Ship Revenue, We Buy Revenue, Case별 건수(C1~C5)
활용 방법: 특정 배송사/국가/기간을 필터링한 후 단가를 조정하여 가격 변경 시나리오에서 예상 배송 매출을 즉시 산출할 수 있습니다.

사용 데이터 상세
shipping_fee_data.json: dashboard_data.json의 패키지 데이터에서 배송비 관련 필드를 추출·집계한 파일입니다.
필드원천 / 설명

spn: carrier, carrier_service

spn: package_weight

spfn: shipping_fee

spn / t: trans_at_utc
final_net_amount (We Buy Revenue)구매대행 최종 결제 금액 (USD)

파생 컬럼
시뮬레이션 계산식:
Ship Revenue = Σ (P_DK × Q_DK) (무게 구간별 합산)
P_DK: 사용자가 직접 수정 가능한 단가 (기본값: 실제 결제 단가 평균)
Q_DK: 해당 무게 구간 패키지 수 (수정 가능)

페이지 개요
2025년 실적을 기준으로 산정한 2026년 회원 등급 결과를 보여주는 페이지입니다.
  • 등급 요약 카드 (5개) — VIP / Gold / Silver / Bronze / Basic 각 인원 수
  • 등급 기준 — 전체 사용 금액 + 배송 횟수 5회 이상 조건 기반 5단계 등급
  • Technical Notes — 2025-03-31 이전 가입 + y25 등급 결측 → Basic 자동 부여 처리 내역
  • 2026 vs 2025 등급 변화 — 각 등급별 2026년 인원 수 + 2025년 대비 증감(+/-)
  • 등급별 평균 실적 — 등급별 인원 수, 전체 사용 금액 평균, 구매대행 결제 평균, 배송비 결제 평균, 창고 옵션 결제 평균
  • 전체 고객 테이블 — Suite Number(Customer Search 링크), 수치별 데이터, 2026~2023 등급 이력
